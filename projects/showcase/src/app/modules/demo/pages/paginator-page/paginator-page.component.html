Paginator Style
A styling class is created to modify the look of ng-bootstrap paginator.

An example of paginator-demo 

        <ngb-pagination [collectionSize]="70" aria-label="Default pagination"></ngb-pagination>

An example of paginator-demo with custom icons 

        
            <ngb-pagination [collectionSize]="70" aria-label="Default pagination">
                <ng-template ngbPaginationPrevious><i class="fa fa-angle-left"></i></ng-template>
                <ng-template ngbPaginationNext><i class="fa fa-angle-right"></i></ng-template>
            </ngb-pagination>
Icon Integration
Icon classes from third party packs like font awesome, Fontello, etc can be used with the paginator for next and previous buttons <i> tag.

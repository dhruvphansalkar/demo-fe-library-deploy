Tree Menu Component
This component will show a hierarchical structure in a tree view. It will internally call itself recursively if the elements passed as input will also contain children, also, it will display an icon whether the item will be expanded or not. Could contains route inside items and even external link.
Input Properties
Property Name Type Description
menuItems TreeMenuModel[] The hierarchical data to display
TreeMenuModel
Property Name Type Description
text string The name of the item that will be displayed
routerLink string Not required. The route of the item if contained.
externalLink string Not required. Link to an external resourcer
children TreeMenuModel[] Not required. Children that will be render as a sublevel of the menu

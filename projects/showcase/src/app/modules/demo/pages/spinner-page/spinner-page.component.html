Spinner Component
This spinner component will show a spinner at the center of the view while preventing the user's interactions with the content on the page.

Please click the button to display the spinner 

          <lib-spinner [showSprinner]="showSpinner" [customClass]="'text-primary'"></lib-spinner>
Input Properties

To control the displaying and hinding of the spinner toggle [showSprinner] input attribute.

To set the size of the spinner use the [size] attribute. The default value is 5.

To assign color to the spinner use the [customClass] attribute. It works with default bootsrap color classes as well as custom css classes.

Service Overview

The showcase of the spinner is controlled using the SpinnerStatusService based on the value of the input property [showSprinner]. The service uses 2 mehods show() and hide() to manage the status of the spinner. 

        
          show(): void {{'{'}}
            document.body.style.position = 'fixed';
            document.body.style.overflow = 'hidden';
            this.loaderSubject.next(true);
          {{'}'}}
        
          hide(): void {{'{'}}
            document.body.style.position = '';
            document.body.style.overflow = '';
            this.loaderSubject.next(false);
          {{'}'}}

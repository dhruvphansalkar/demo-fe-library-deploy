Date Picker Component
This components provide a DatePicker based on ng-bootstrap's one. It can be easily integrated inside forms and could be used even as an input element.
How to use
For using the DatePicker we need to import the DatePickerModule. The module provides the DatePickerComponent and the DatePickerConfig. It could be placed inside a Form for working properly and a FormControl should be passed to it for accessing all the FormControl functionalities such as provide disable/enable API, or used with a raw model using ngModel A DatePickerConfig object could be passed for configuring the DatePicker, otherwise it will have a default behaviour. You could size the input passing, or wrapping, the component with some Bootstrap row and col classes

Input properties
Property Name Type Description
id string Used for passing an id, if null 'date-picker' will be used as default
label string Used for passing a label, if null 'Date Picker' will be used as default
footerText string Used for passing the text to display as button for selecting the today date, if null will be used 'today' as default
config DatePickerConfig Used for passing some configurations to DatePickerComponent, see the below section for further information
Output events
Property Name Type Description
todayClick NgbDate Event emitter for handling the today click
DatePickerConfig
Property Name Type Description
showLabel boolean Used for displaying label
placeholder String Placeholder string
navigation string Used for placing the date picker
startDate NgbDate Used for setting an initial value
autoclose true, false, 'inside', 'outside' Used handling the autoclose

Input Text Directive
This is a directive to render every text-input in Demo product

An example of input text 

        <input type="text" libInputForm placeholder="Write here something">

An example of input text disabled 

      <input type="text" libInputForm placeholder="Write here something" disabled >

To stylize the input text as showed in examples add the form-control-demo class to the input HTML element.

To disable the the input text as showed in examples add the disabled attribute to the input HTML element.

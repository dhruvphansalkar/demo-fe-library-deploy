Nav
This is a list of a links navigating to specified location. They are usually presented as a horizontal bar of links at the top of every page.

An example of Nav

HTML
          
            
              <lib-nav [navMenu]=navMenu></lib-nav>
TS
        
    
            navMenu: object = [
                {{'{'}} label: 'Vendite Attive', path: '#' {{'}'}},
                {{'{'}} label: 'Opportunity', path: '#' {{'}'}},
                {{'{'}} label: 'Vendite Estine', path: '#' {{'}'}}
            ];
Input Properties

To add a nav menu, add the label and path of the new menu in the navMenu object of component.ts file and bind navMenu input in the selector tag of lib-nav as shown in the example.

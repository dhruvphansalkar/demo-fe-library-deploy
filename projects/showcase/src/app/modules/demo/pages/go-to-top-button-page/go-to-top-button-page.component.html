Go to top Button Component
This component will show a button when scrolling is performed and go to top of the page upon clicking it.

An example of input text 

        <lib-go-to-top-button></lib-go-to-top-button>
Input Properties

To assign a id to the button use [id] input attribute.

To assign a perticular scroll height at which the button becomes visible [scrollHeight] input attribute.

To assign custom classes to the button use [customClass] input attribute.

API Methods

To display the button use showButton(): void method.

To hide the button use hideButton(): void method.

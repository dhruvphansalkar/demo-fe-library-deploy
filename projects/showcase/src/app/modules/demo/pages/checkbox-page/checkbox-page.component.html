Checkbox
Here we have two checkboxes with different look and feel.

An example of Custom Bootstrap Checkbox 

    
        <div libCheckboxContainer>
            <input type="checkbox" libCheckboxForm id="customCheck1">
            <label libCheckboxLabel for="customCheck1">Custom bootstrap checkbox</label>
        </div>

An example of Custom Bootstrap Checkbox disabled 

    
        <div libCheckboxContainer>
            <input type="checkbox" libCheckboxForm id="customCheck1" disabled>
            <label libCheckboxLabel for="customCheck1">Custom bootstrap checkbox</label>
        </div>

To stylize the input checkbox as showed in examples add all the classes to the HTML element.

To disable the input checkbox as showed in examples add the disabled attribute to the input HTML element.

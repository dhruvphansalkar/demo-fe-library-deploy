Accordion Component
This is a accordion with 2 states controlled by the expanded property

An example of an Accordion

accordian header content
accordion content 
      
      <lib-accordion [expanded] = "expanded">
        <div header-content>
        accordian header content
        </div>
        <div body-content>
          Accordian Content
        </div>
      </lib-accordion>
Input Properties

To control the openening and closing of the accordion use [expanded] attribute.

Methods

To toggle the state of the accordian use toggle(): boolean method.

To close the accordian use closeAccordion(): void method.

To open the accordian use openAccordion(): void method.

To get the status of accordian's expansion use isExpanded(): boolean method.

Content Selectors

To customize or add content to the accordions header use header-content.

To add content to the accordions body use body-content.

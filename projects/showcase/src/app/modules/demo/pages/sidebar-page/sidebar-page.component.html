Sidebar
Sidebar is sticky and is collapsible to the left side. It works via content trasclution. A section named [input] could be passed for render a input text for searching and another section maed [menu] could be passed for rendere the menu, above I used the Tree Menu Component.

An example of Sidebar
This is how the main page should be structured.
import {{ "{" }} SidebarModel {{ "}" }} from 'projects/lib/src/lib/modules/sidebar/sidebar'; 
      
          sidebarItems: Array<SidebarModel> = [
              {{'{'}} text: 'Input Text', routerLink: '/demo/input-text-page'{{'}'}},
              {{'{'}} text: 'Buttons', route: '/demo/button-examples'{{'}'}},
              ];
NgContent Sections
[input] This section is used for creating sidebar with an Input Text at the top
[menu] This section is used for creating sidebar with a menu

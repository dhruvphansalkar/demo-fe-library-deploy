Switch
Switch is used for activating one of two predefined options.

An example of Switch

one 
        
            <div class="custom-control-switch-demo custom-switch-demo">
            <span class="custom-label-demo">one</span>
            <input type="checkbox" class="custom-control-switch-input-demo" id="customSwitch1"
                (change)="onSelectionChange($event.target.checked)">
            <label class="custom-control-switch-label-demo" for="customSwitch1">two</label>
            </div>

To stylize the switch as showed in examples add all the classes to the HTML element.

To disable the switch as showed in examples add the disabled attribute to the input HTML element.

DropDown Component
This is a dropdown input component which implements ControlValueAccessor interface to act as a form control.

An example of Drop-Down component

Liverpool United Chelsea

------------------HTML------------------

        
          <lib-dropdown [defaultLabel] = "'Select your club'" >
            <lib-dropdown-option [value] = "'liv'">Liverpool</lib-dropdown-option>
            <lib-dropdown-option [value] = "'utd'">United</lib-dropdown-option>
            <lib-dropdown-option [value] = "'che'">Chelsea</lib-dropdown-option>
          </lib-dropdown>
Properties (lib-dropdown)

To disable the control using the template use [isDisabled] input property.

To pass a default label to the dropdown use [defaultLabel] input property. Note that this does not assign value to the dropdown.

To set the dropdown button at a block level set [blockButton] input property.

To customize the placement of the dropdown list use [placement] input property. The default value is bottom-left.

To detect the change in value of the dropdown use (valueChange) output property.

Properties (lib-dropdown-option)

To assign a value to the option use [value] input property (required).

(lib-dropdown)

To assign a on load value to the form control use DropdownModel and pass during formControl initialization.

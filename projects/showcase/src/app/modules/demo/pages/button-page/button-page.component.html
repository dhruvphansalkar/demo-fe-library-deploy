Button Styles
Multiple styling classes are created by extending existing bootsrap classes to create new look buttons

An example of btn-primary-demo 

        <button class="btn btn-primary-demo">Enabled primary button</button>

An example of btn-primary-demo disabled 

        <button disabled class="btn btn-primary-demo">Disabled primary button</button>

An example of btn-success-demo 

        <button class="btn btn-success-demo">Enabled success button</button>

An example of btn-success-demo disabled 

        <button disabled class="btn btn-success-demo">Disabled primary button</button>

An example of btn-outline-primary-demo 

        <button class="btn btn-outline-primary-demo">Outline Primary</button>

An example of btn-outline-primary-demo disabled 

        <button disabled class="btn btn-outline-primary-demo">Outline Primary</button>

An example of btn-link-demo 

        <button class="btn btn-link-demo m-1">IMPORTA DA XML</button>

An example of btn-link-demo disabled 

        <button disabled class="btn btn-link-demo m-1">IMPORTA DA XML</button>
Icon Integration
Icon classes from third party packs like font awesome, Fontello, etc can be used with these classes just like with bootstrap buttons. They can be included by applying third party icon classes to the html <i> tag.
Button Sizing
Bootstrap button sizing classes also work with library button using .btn-block, .btn-lg, etc classes as shown in the demo

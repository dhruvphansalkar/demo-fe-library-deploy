Pill Tab Set Component
This is a tabs component which changes view based on which tab is selected

An example of pills-tab-component

Content for Tab 1 Content for Tab 2 Content for Tab 3 
        
        <lib-pill-tab-set>
          <lib-pill-tab [selectedColor] = "'#84BF41'" [title]="'Tab 1'">
            Content for Tab 1
          </lib-pill-tab>
          <lib-pill-tab [title]="'Tab 2 '">
            Content for Tab 2  
          </lib-pill-tab>
          <lib-pill-tab [title]="'Tab 3'">
            Content for Tab 3               
          </lib-pill-tab>
        </lib-pill-tab-set>

To set the color of selected pill use the [selectedColor] attribute on <lib-pill-tab-set> selector.

To set a tab to be selected on load use the [selectedTabIndex] input attribute and pass the index of the tab

To set the title of each pill use the [title] attribute on each <lib-pill-tab-set> selector.

Use the (tabClickEvent) emmiter to detect tab click event, it will emit the index of the tab clicked.

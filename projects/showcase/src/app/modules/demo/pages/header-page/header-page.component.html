Header
The Header consists of a hamburger menu, company logo, company description, language selection and operator details

An example of Header

Google Operator[12345] Dhruv - 52003 - Google[0001] 
        
          <lib-header>
            <img companyLogo src="" class="img-fluid">
            <a companyDescription>Google</a>
            <span operatorDetails>Operator[12345] Dhruv Phansalkar - 52003 - Google[0001]</span>
          </lib-header>

To change company logo, as shown in example along with companyLogo put proper image source inside the selector tag of lib-header.

As shown in example along with companyDescription add company description.

As shown in example along with operatorDetails add operator details.

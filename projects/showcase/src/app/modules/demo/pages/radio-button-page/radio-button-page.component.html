Radio Button Directive
Radio Button is used for activating one of two predefined options.

An example of Radio Button 

        
            <div class="container">
                <div libRadioContainer>
                    <input type="radio" id="Codice" name="Ordina" value="Codice" libRadioForm
                        (click)="selectedValue($event.target.value)">
                    <label libRadioLabel for="Codice">Codice </label>
                </div>
            
                <div libRadioContainer>
                    <input type="radio" id="Descrizione" name="Ordina" value="Descrizione" libRadioForm
                         (click)="selectedValue($event.target.value)">
                    <label libRadioLabel for="Descrizione">Descrizione </label>
                </div>
            </div>

To stylize the radio button as showed in examples add all the classes to the HTML element.

To disable the radio button add the disabled attribute to the input HTML element.
